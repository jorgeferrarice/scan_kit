## 0.1.0

- Initial release
- LiDAR scanning mode using ARKit mesh anchors
- TrueDepth (face) scanning mode using front-facing depth camera
- Real-time mesh streaming with configurable update frequency
- Mesh classification support (wall, floor, ceiling, table, seat, window, door)
- OBJ file export
- GLB (binary glTF) file export
- On-demand mesh snapshots
